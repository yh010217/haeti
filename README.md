# 🌅해티프로젝트  : 교사 대상 온라인 중고거래 웹 서비스 





## 프로젝트 기간



2024.04.16~2024.04.28 (총 13일)
천재교육풀스택 해티조 

|김보경(팀장)|은별|최용훈|홍유나|
|------|------|------|------|
|상품구매<br> 관리자페이지|마이페이지|상품판매<br>채팅|회원가입<br> 로그인|


## 기획의도
+ 교육 관련 자료를 효율적으로 거래할 수 있는 공간을 마련하여 교육 현장에서 필요한 물품을 가까운 곳에서 빠르게 거래 할 수 있도록 동네 기반의 중고 직거래 플랫폼 제작
+ 같은 직업군 내에서 거래를 함으로써 필요한 자료의 가치와 경험 공유가 가능
+ 사용자가 물품을 직접 확인하고 거래를 완료하도록 하여 사기 발생 가능성을 최소화 

## 주요 특징
+ 구매를 희망하는 이용자가 거래를 진행할 수 있도록 1대 1채팅 기능 구현
+ 주소 API를 이용해 주소 정보를 표준화된 형식으로 저장하고 활용
+ 관심지역의 매물을 지도와 함께 확인 할 수 있는 매물지도 메뉴를 구성
